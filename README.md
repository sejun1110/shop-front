## Overview

이 프로젝트는 단순 쇼핑몰이 아니라  
ERP / MES와 연결되는 주문 생성 시스템으로 설계되었습니다.

사용자는 상품을 조회하고, SKU 옵션을 선택하고,  
장바구니 및 주문을 생성하며 이후 ERP 재고 흐름으로 연결됩니다.

---

## Key Features

- 상품 목록 / 상세 조회
- SKU 옵션 선택
- 재고 기반 수량 선택
- 장바구니 관리
- 주문 생성
- 주문 목록 / 상세 조회
